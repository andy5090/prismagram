- [ ] See the feed
- [ ] Like / Unlike a photo
- [ ] Comment on a photo
- [ ] Search by user
- [ ] Search by location
- [ ] See user profile
- [ ] Follow / Unfollow User
- [ ] See the full photo
- [ ] Edit my profile
- [*] Create account
- [ ] Log in
- [ ] Upload a photos
- [ ] Edit the photo (Delete)
